mypassion

I grew up around computers. You could even say it’s in my genes. My grandpa worked on computers on the railroad and my dad has been the system admin at our local hospital for about 20 years. When I was 8 years old I built my first website all about Pokémon. Just another stereotypical geeky 90s kid.

Building sites was just a hobby until 2014 when I was hired on as a junior developer for an ad agency. I quickly moved on to lead developer and have been with the company ever since. For the past 6 years I primarily developed for Wordpress but I’m always looking to branch out and have recently been working with React.

webportfolio

YelpCamp

This site was built as class project. Although not for an actual client, this allow for many features that would be required for a true production site.

It allows for:

  • User Signup using PassportJS and stored in MongoDB
  • Assignment of admin roles
  • Adding/removing campsites by users only
  • Adding/removing comments by users only
  • Public user profiles
  • Bootstrap layout

Joe Brown Attorney

Launched in July 2020, this is a simple website to help a returning lawyer. The site is very modern and is heavily focused on dark grays and blues.

It's built with:

  • Wordpress
  • Community plugins including: Advanced Custom Fields, Gravity Forms, and GDPR Compliance
  • Utilizes Custom Post Types to allow client to easily make content edits
  • Google Maps integration
  • Bootstrap layout
